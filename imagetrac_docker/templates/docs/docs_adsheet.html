Adding an Adsheet

These are the instructions for processing individual adsheet files:

The basic steps are as follows:

  • rename/place the .xlsx file
  • run the adsheet.py format utility
  • run the import_adsheet navicat scheduled import job

prepare the file

Rename the dated xlsx file to "adsheet.xlsx" (eg. "8-AUGUST 14-2016.xlsx" becomes "adsheet.xlsx". Move the file to ADVERTISING/Inserts/MAGETRAC/csv (overwriting the existing one in this directory).

run the format utility

from the terminal (shell) on the server workstion, run adsheet.py. When prompted, enter the ad date in the DDMMYY format.  
Then (in CAPS), enter the ad version (PRE PRE, PRE or FINAL)  

importer

Find the "import_adsheet" task in navicat. Double click the icon, then press "start" to initiate the task.
