Updating Inventory Data

These are the instructions for processing "the daily photo audit file":

The basic steps are as follows:

  • convert the .xlsx file to csv
  • add utf-8 encoding to the csv file
  • run the quantities.py format utility
  • run the update_inventory navicat scheduled import job

necesary software:

  • MS Excel
  • Sublime Text (or similar code editor)

convert the file

Open the daily photo audit file in Excel. Save the file in the windows comma seperated values (csv) format; name it "universe_wcsv.csv", and save it to ADVERTISING/Inserts/MAGETRAC/csv (overwriting the existing one in this directory).

add utf-8 encoding

open the newly created "universe_wcsv.csv" in Sublime Text. Simply resave the file with UTF-8 encoding:

quantities.py

from the terminal (shell) on the server workstion, run quantities.py

importer

Find the "update_inventory" task in navicat. Double click the icon, then press "start" to initiate the task.
