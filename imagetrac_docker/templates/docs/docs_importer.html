Creating a Navicat Import Task

These are the instructions for creating/modifying a navicat importer. An importer is a task that maps spreadsheet data to a database table.

The basic steps are as follows:

  • convert the .xlsx file to csv
  • add utf-8 encoding to the csv file
  • create the navicat task
  • configure the navicat task

convert the file

Open your target spreadsheet in Excel. Save the file in the windows comma seperated values (csv) format, to a temporary holding directory (e.g. ADVERTISING/Inserts/MAGETRAC/csv/ or, if on the server workstation - /Users/imagetracdb/b5crontab/temp/).

add utf-8 encoding

open the newly created csv file in Sublime Text. Simply resave the file with UTF-8 encoding:

create the importer

from the tables screen in navicat, click "Import" on the footer.  

configure the importer

though you have a choice of file formats, normally you will want to use csv click "Add File" and navigate to your csv click "continue" uncheck 4 digit years choose the database table to which you'll be adding data for each table field, indicate the corresponding csv data column choose your preferred import behavior choose "Save as" to name yout task; "Save" to simply save a modification; or "Start" to run your task.
